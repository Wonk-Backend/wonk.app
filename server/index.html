https://wonk.app/server/ is used for all functions involving a server. We use Wonk Microserver to run all functions on your computer directly which has multiple benefits:

  • Faster connection times
  • Increased security as data does not leave your local network
  • Reduced latency
  • Better performance due to local processing
  • Offline capabilities

This page includes all information about the website and your microserver.
